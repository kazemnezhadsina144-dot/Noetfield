Partner standards (non-negotiables)

These rules protect delivery quality and keep outputs defensible.

  • Public = non-confidential: no secrets/credentials/regulated personal data in forms.
  • Named artefacts: memo/controls/evidence index formats must stay consistent.
  • Evidence IDs: claims need a traceable reference (export, screenshot, policy, log).
  • Red-lines: escalation path when blockers appear (privacy/security/procurement).
