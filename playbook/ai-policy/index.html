What is included

A compact policy system designed to be adopted, not ignored.

Policy + guardrails

  • Responsible AI Use Policy (editable template).
  • Role guardrails: Legal / HR / Product / IT / Security (editable).
  • Prompt + data handling rules (confidentiality-first).
  • Tool classification: allowed / restricted / prohibited.

Rollout artefacts

  • Quick reference sheet (one-page summary).
  • Sign-off tracking template (lightweight).
  • Incident guidance (what to do when something goes wrong).

When to use this kit

Best for organizations that already have AI usage and need a defensible baseline quickly.

Typical triggers

  • AI usage is happening informally across teams.
  • Executives are asking for “clear rules” and accountability.
  • Procurement and security need consistent handling for tools.
