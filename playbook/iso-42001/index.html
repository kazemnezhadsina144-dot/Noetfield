What is included

A structured set of templates to make AIMS readiness implementable.

Scope + boundaries

  • Scope statement template (systems, use-cases, exclusions).
  • Roles and responsibilities template.
  • Boundary and dependency mapping worksheet.

Controls-to-evidence map

  • Controls mapping table (owner, cadence, evidence hook).
  • Evidence index template (what exists, where it lives, freshness).
  • Exceptions + compensating controls worksheet.

Executable evidence plan

  • 90-day evidence capture plan template (cadence + milestones).
  • Review rhythm and sign-off template.
  • Audit request readiness checklist.

How to use this pack

A minimal operating flow to turn templates into a working readiness baseline.

Suggested sequence

  • Lock scope and boundaries first.
  • Map controls to owners and evidence hooks.
  • Implement the evidence plan cadence and reviews.
