What is included

A practical, repeatable evaluation workflow designed to reduce back-and-forth and improve decision defensibility.

Core templates

  • Pass/fail gates (minimum standards) checklist.
  • Weighted scoring rubric (risk-based, adjustable weights).
  • Evidence request list (vendor pack requirements).
  • Decision memo template (board/audit-friendly).

Operating guidance

  • How to run the evaluation in shortlisting vs contract review vs renewal.
  • How to classify data exposure and set conditions for approval.
  • How to document exceptions without oversharing sensitive details.

Who this is for

Teams that need a clean vendor decision record under governance and audit pressure.

Primary users

  • Procurement / Vendor Management
  • Security / IT / Risk / Compliance
  • Legal (as an input, not a replacement)

Common triggers

  • New AI vendor shortlist, fast cycle, unclear requirements.
  • Contract review with data-use and training ambiguity.
  • Renewal where auditability and evidence is now required.
